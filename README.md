# Chronos Pomodoro

Aplicação de timer Pomodoro desenvolvida com React, TypeScript e CSS Modules.

## Tecnologias

- [React](https://react.dev/) — biblioteca para construção de interfaces
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- CSS Modules — estilização com escopo por componente

## Como Executar

```bash
npm install
npm run dev
```

## Deploy

[Chronos Pomodoro](https://chronos-pomodoro-three-eta.vercel.app)

## Sobre

Projeto desenvolvido para praticar React com TypeScript, aplicando componentização, tipagem e gerenciamento de estado.
