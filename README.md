# Chronos Pomodoro

Aplicação de timer Pomodoro desenvolvida com React, TypeScript e CSS Modules, com deploy na Vercel.

## Tecnologias

- [React](https://react.dev/) — biblioteca de interface
- [TypeScript](https://www.typescriptlang.org/) — tipagem estática
- [Vite](https://vitejs.dev/) — bundler e servidor de desenvolvimento
- CSS Modules — estilização com escopo por componente

## Como Executar

Requisito: [Node.js](https://nodejs.org/)

```bash
# Instalar dependências
npm install

# Iniciar o servidor
npm run dev
```

Acesse: `http://localhost:5173`
